# WhatsTauri v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A pure-Rust Tauri 2 (2.11.x) Linux-only wrapper around web.whatsapp.com with persistent login, tray + close-to-tray, native click-to-focus notifications, blob downloads, crash auto-reload, and RPM packaging.

**Architecture:** No frontend — the main window loads the remote URL with a Chrome user-agent. Two injected init-scripts bridge notifications and the window title (unread count) to Rust over three whitelisted IPC commands. Everything else (downloads, link policy, tray, crash recovery, data wipe) is engine-side Rust. Spec: `docs/superpowers/specs/2026-06-11-whatstauri-design.md`.

**Tech Stack:** Tauri 2.11.x (`tray-icon` feature), wry/WebKitGTK (webkit2gtk-4.1), `notify-rust` (D-Bus notifications with actions), `tauri-plugin-single-instance`, `tauri-plugin-opener`, `tauri-plugin-dialog` (wipe confirmation), `serde`+`toml`, `dirs`.

use serde::{Deserialize, Serialize};
use std::path::PathBuf;

/// Bump the Chrome version here each release (spec §6.3 "UA rot").
const DEFAULT_UA: &str = "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 \
(KHTML, like Gecko) Chrome/138.0.0.0 Safari/537.36";

#[derive(Clone, Debug, Serialize, Deserialize)]
#[serde(default)]
pub struct Config {
    pub user_agent: String,
    pub close_to_tray: bool,
    pub start_hidden: bool,
    /// Sets WEBKIT_DISABLE_DMABUF_RENDERER=1 before webview init (NVIDIA/Wayland glitches).
    pub disable_dmabuf_renderer: bool,
    /// Sets WEBKIT_DISABLE_COMPOSITING_MODE=1 before webview init.
    pub disable_compositing: bool,
}

impl Default for Config {
    fn default() -> Self {
        Self {
            user_agent: DEFAULT_UA.to_string(),
            close_to_tray: true,
            start_hidden: false,
            disable_dmabuf_renderer: false,
            disable_compositing: false,
        }
    }
}

impl Config {
    pub fn from_toml(s: &str) -> Self {
        toml::from_str(s).unwrap_or_else(|e| {
            eprintln!("whatstauri: invalid config, using defaults: {e}");
            Self::default()
        })
    }

    pub fn config_path() -> Option<PathBuf> {
        dirs::config_dir().map(|d| d.join("whatstauri").join("config.toml"))
    }

    /// Missing file or unreadable file → defaults. Never refuses to start (spec §3.4).
    pub fn load() -> Self {
        match Self::config_path().and_then(|p| std::fs::read_to_string(p).ok()) {
            Some(s) => Self::from_toml(&s),
            None => Self::default(),
        }
    }

    /// Must run BEFORE any GTK/WebKit initialization.
    pub fn apply_env_workarounds(&self) {
        if self.disable_dmabuf_renderer {
            std::env::set_var("WEBKIT_DISABLE_DMABUF_RENDERER", "1");
        }
        if self.disable_compositing {
            std::env::set_var("WEBKIT_DISABLE_COMPOSITING_MODE", "1");
        }
    }
}
```

/// Extract unread count from WhatsApp Web's document title ("(3) WhatsApp").
/// Unrecognized titles return `previous` — never wipe a real count on noise.
pub fn parse_unread(title: &str, previous: u32) -> u32 {
    let t = title.trim();
    if let Some(rest) = t.strip_prefix('(') {
        if let Some(end) = rest.find(')') {
            if let Ok(n) = rest[..end].parse::<u32>() {
                return n;
            }
        }
        return previous;
    }
    if t.contains("WhatsApp") {
        0
    } else {
        previous
    }
}
```

/// In-app navigation allowlist (spec §3.1): *.whatsapp.com / *.whatsapp.net, https only.
pub fn is_internal(url: &url::Url) -> bool {
    if url.scheme() != "https" {
        return false;
    }
    matches!(url.host_str(), Some(h) if h == "whatsapp.com"
        || h.ends_with(".whatsapp.com")
        || h == "whatsapp.net"
        || h.ends_with(".whatsapp.net"))
}

/// The "browser not supported" wall (spec §5.1).
pub fn is_browser_wall(url: &url::Url) -> bool {
    is_internal(url) && url.path().ends_with("/browsers.html")
}

```javascript
// Replaces window.Notification so WhatsApp's notifications become native ones
// delivered by Rust over IPC (spec §3.2). Runs before any page script.
(function () {
  "use strict";

  function invoke(cmd, args) {
    var i = window.__TAURI_INTERNALS__;
    if (i && typeof i.invoke === "function") {
      i.invoke(cmd, args).catch(function () {});
      return true;
    }
    return false;
  }

  // IPC bootstrap may land after us — queue until it exists.
  var queue = [];
  function send(cmd, args) {
    if (!invoke(cmd, args)) queue.push([cmd, args]);
  }
  var drain = setInterval(function () {
    while (queue.length && invoke(queue[0][0], queue[0][1])) queue.shift();
    if (window.__TAURI_INTERNALS__ && !queue.length) {
      clearInterval(drain);
      send("debug_log", { message: "notification-shim: IPC ready" });
    }
  }, 100);

  function FakeNotification(title, opts) {
    opts = opts || {};
    send("deliver_notification", {
      title: String(title || ""),
      body: String(opts.body || ""),
    });
  }
  FakeNotification.permission = "granted";
  FakeNotification.requestPermission = function (cb) {
    if (cb) cb("granted");
    return Promise.resolve("granted");
  };
  // Instance API surface WhatsApp may touch — all no-ops.
  FakeNotification.prototype.close = function () {};
  FakeNotification.prototype.addEventListener = function () {};
  FakeNotification.prototype.removeEventListener = function () {};

  Object.defineProperty(window, "Notification", {
    value: FakeNotification,
    writable: false,
    configurable: false,
  });

  send("debug_log", { message: "notification-shim: installed" });
})();
```

- [ ] **Step 2: Create `src-tauri/inject/title-watcher.js`**

```javascript
// Ships the raw document title to Rust (parsing happens Rust-side, spec §3.3)
// and holds a WebLock so WebKitGTK doesn't over-throttle the hidden page (§6.5).
(function () {
  "use strict";

  function invoke(cmd, args) {
    var i = window.__TAURI_INTERNALS__;
    if (i && typeof i.invoke === "function") {
      i.invoke(cmd, args).catch(function () {});
      return true;
    }
    return false;
  }

  // Throttling insurance: a held lock keeps the page schedulable while hidden.
  if (navigator.locks && navigator.locks.request) {
    navigator.locks.request("whatstauri-keepalive", function () {
      return new Promise(function () {}); // held forever
    });
  }

  var lastSent = null;
  function tick() {
    var t = document.title;
    if (t !== lastSent && invoke("report_title", { title: String(t) })) {
      lastSent = t;
    }
  }
  setInterval(tick, 1000); // polling beats MutationObserver here: <title> may not exist yet at init-script time

  var announce = setInterval(function () {
    if (invoke("debug_log", { message: "title-watcher: attached" })) clearInterval(announce);
  }, 100);
})();
```

```rust
use std::sync::Mutex;
use tauri::{AppHandle, Manager};

/// Last unread count — report_title keeps previous value on unparseable titles.
pub struct UnreadState(pub Mutex<u32>);

/// Web Notification → native notification with click-to-focus (spec §3.2).
/// notify-rust's D-Bus "default" action fires when the notification body is clicked.
#[tauri::command]
pub fn deliver_notification(app: AppHandle, title: String, body: String) {
    std::thread::spawn(move || {
        let shown = notify_rust::Notification::new()
            .summary(&title)
            .body(&body)
            .appname("WhatsTauri")
            .action("default", "Open")
            .show();
        match shown {
            Ok(handle) => handle.wait_for_action(|action| {
                if action == "default" {
                    focus_main(&app);
                }
            }),
            Err(e) => eprintln!("whatstauri: notification failed: {e}"),
        }
    });
}

#[tauri::command]
pub fn report_title(app: AppHandle, title: String) {
    let state = app.state::<UnreadState>();
    let mut last = state.0.lock().unwrap();
    let n = crate::unread::parse_unread(&title, *last);
    if n != *last {
        *last = n;
        crate::tray::set_unread(&app, n);
    }
}

/// Init-script lifecycle logging, active only with --debug (spec §5.5).
#[tauri::command]
pub fn debug_log(message: String) {
    if std::env::args().any(|a| a == "--debug") {
        eprintln!("[whatstauri:init] {message}");
    }
}

pub fn focus_main(app: &AppHandle) {
    if let Some(w) = app.get_webview_window("main") {
        let _ = w.show();
        let _ = w.unminimize();
        let _ = w.set_focus();
    }
}
```

use tauri::webview::{DownloadEvent, WebviewWindowBuilder};
use tauri::{AppHandle, WebviewUrl, WebviewWindow};

const WHATSAPP_URL: &str = "https://web.whatsapp.com";

pub fn build_main_window(app: &AppHandle, cfg: &crate::config::Config) -> tauri::Result<()> {
    let url: url::Url = WHATSAPP_URL.parse().expect("static url");
    let nav_app = app.clone();
    let dl_app = app.clone();

    let win = WebviewWindowBuilder::new(app, "main", WebviewUrl::External(url))
        .title("WhatsTauri")
        .inner_size(1100.0, 750.0)
        .visible(!cfg.start_hidden)
        .user_agent(&cfg.user_agent)
        .disable_drag_drop_handler() // let HTML5 file drag-and-drop reach WhatsApp (spec §3.1)
        .initialization_script(include_str!("../inject/notification-shim.js"))
        .initialization_script(include_str!("../inject/title-watcher.js"))
        .on_navigation(move |url| {
            if crate::links::is_browser_wall(url) {
                eprintln!(
                    "whatstauri: WhatsApp served the unsupported-browser page. \
                     Remedies: bump user_agent in config.toml, or tray > Clear data & re-login."
                );
                let _ = notify_rust::Notification::new()
                    .summary("WhatsTauri: browser check failed")
                    .body("WhatsApp rejected the current user agent. Edit user_agent in \
                           config.toml or use 'Clear data & re-login' from the tray.")
                    .appname("WhatsTauri")
                    .show();
                return true; // still show the page; it names what's missing
            }
            if crate::links::is_internal(url) {
                return true;
            }
            let _ = tauri_plugin_opener::open_url(url.as_str(), None::<String>);
            let _ = nav_app; // keep handle alive for future use
            false
        })
        .on_download(move |_webview, event| {
            match event {
                DownloadEvent::Requested { destination, .. } => {
                    // WebKitGTK destination override is historically flaky (spec §3.1);
                    // if downloads stall, fall back to moving the file in Finished.
                    if let Some(dir) = dirs::download_dir() {
                        let name = destination
                            .file_name()
                            .map(|s| s.to_os_string())
                            .unwrap_or_else(|| "whatsapp-download".into());
                        *destination = dir.join(name);
                    }
                }
                DownloadEvent::Finished { path, success, .. } => {
                    let body = match (success, path) {
                        (true, Some(p)) => format!("Saved {}", p.display()),
                        (true, None) => "Download finished".to_string(),
                        _ => "Download failed".to_string(),
                    };
                    let _ = notify_rust::Notification::new()
                        .summary("WhatsTauri download")
                        .body(&body)
                        .appname("WhatsTauri")
                        .show();
                    let _ = &dl_app;
                }
                _ => {}
            }
            true
        })
        .build()?;

    install_crash_recovery(&win);
    Ok(())
}

/// Auto-reload on WebKitGTK web-process crash, max 3 per minute (spec §5.2).
/// Tauri doesn't expose the signal; the raw webkit2gtk handle does.
fn install_crash_recovery(win: &WebviewWindow) {
    #[cfg(target_os = "linux")]
    {
        use std::cell::RefCell;
        use std::rc::Rc;
        use std::time::{Duration, Instant};

        let _ = win.with_webview(|platform_webview| {
            use webkit2gtk::WebViewExt;
            let wv = platform_webview.inner();
            let crashes: Rc<RefCell<Vec<Instant>>> = Rc::new(RefCell::new(Vec::new()));
            wv.connect_web_process_terminated(move |wv, reason| {
                eprintln!("whatstauri: web process terminated ({reason:?})");
                let now = Instant::now();
                let mut c = crashes.borrow_mut();
                c.retain(|t| now.duration_since(*t) < Duration::from_secs(60));
                c.push(now);
                if c.len() <= 3 {
                    wv.reload();
                } else {
                    let _ = notify_rust::Notification::new()
                        .summary("WhatsTauri keeps crashing")
                        .body("The web view crashed repeatedly. Try setting \
                               disable_dmabuf_renderer = true in config.toml.")
                        .appname("WhatsTauri")
                        .show();
                }
            });
        });
    }
}

use std::path::PathBuf;
use tauri::{AppHandle, Manager};
use tauri_plugin_dialog::{DialogExt, MessageDialogButtons, MessageDialogKind};

fn marker_path(app: &AppHandle) -> Option<PathBuf> {
    app.path().app_config_dir().ok().map(|d| d.join(".wipe-on-start"))
}

/// Tray menu entry: confirm (non-blocking — menu events run on the GTK main
/// thread, where a blocking dialog would deadlock), then mark + restart.
pub fn request_wipe(app: &AppHandle) {
    let app2 = app.clone();
    app.dialog()
        .message(
            "This deletes all local WhatsApp data and logs you out.\n\
             You will need to scan the QR code again.",
        )
        .title("Clear data & re-login")
        .kind(MessageDialogKind::Warning)
        .buttons(MessageDialogButtons::OkCancel)
        .show(move |confirmed| {
            if !confirmed {
                return;
            }
            if let Some(marker) = marker_path(&app2) {
                if let Some(parent) = marker.parent() {
                    let _ = std::fs::create_dir_all(parent);
                }
                if std::fs::write(&marker, b"").is_ok() {
                    app2.restart();
                }
            }
        });
}

/// Runs in setup() BEFORE the webview opens its storage.
pub fn handle_pending_wipe(app: &AppHandle) {
    let Some(marker) = marker_path(app) else { return };
    if !marker.exists() {
        return;
    }
    let _ = std::fs::remove_file(&marker);
    if let Ok(data_dir) = app.path().app_local_data_dir() {
        match std::fs::remove_dir_all(&data_dir) {
            Ok(()) => eprintln!("whatstauri: webview data wiped ({})", data_dir.display()),
            Err(e) => eprintln!("whatstauri: wipe failed: {e}"),
        }
    }
}
